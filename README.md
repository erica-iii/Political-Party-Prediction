# DS4420FinalProject
POLITICAL SPEECH CNN CLASSIFIER
Data: https://data.millercenter.org 
A 1D Convolutional Neural Network built to classify presidential speeches and political quotes as either Democrat or Republican or other party based on textual rhetoric.

Technologies Used:
Python
TensorFlow (Keras)
Scikit-Learn
NumPy

Methods:
The text data is obtained directly from local JSON files, cleaned of HTML tags and punctuation, tokenized, and padded to a maximum length of 500 words.
